Wrap Up

December 31, 2022 · 1 min

After running through the book and getting a fully working feed-forward deep learning API and learning the inner workings of neural networks as a result, I’ve decided not to pursue this particular book any further.

The purpose of reading this particular book was to enable me to get a better understanding into what makes neural networks tick, and as a result it was necessary to write this series of posts with diagrams, maths, and code.

However I feel I don’t need to continue to develop my own API and to understand the mathematics behind rolling my own convolutional neural networks or recurrent neural networks (or any other alternative architecture).

I won’t be using Eidetic for my own projects and instead will opt for a better developed, production ready API like TensorFlow.

I will thus be able to learn about these alternative architectures when using TensorFlow, or when looking into a higher-level machine learning book using one of these higher level libraries.

As such this will be my last post in this category (deep learning from scratch), and any further machine learning will come from alternative sources. “Deep Learning From Scratch” has served its purpose.

Note that Eidetic will still continue to be available but won’t be getting updated.
